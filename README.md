# API Contract Management System - Frontend

A React-based web application for an 'API Management System'. Includes managing contracts, clients, products, and more. This frontend provides an intuitive interface for internal company employees and management to track and manage contracts with clients. 

The backend service: https://github.com/RohanMankame/API-Contract-Management-System

## Prerequisites

- Node.js 16+ and npm
- API backend running on `http://localhost:5000`
